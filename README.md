# rappi-front-test

### How to run

1. Clone this repo
2. Go to the API folder:
```sh
cd rappi-front-test/api
```
3. Install API dependencies
```sh
npm install
```
4. Run API
```sh
node index.js
```
5. Go to the APP folder:
```sh
cd rappi-front-test/app
```
6. Install APP dependencies
```sh
npm install
```
7. Run APP
```sh
npm run dev
```
